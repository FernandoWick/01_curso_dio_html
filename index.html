O que são tags

As tags constituem a linguagem de marcação html. Elas basicamente definem o tipo de cada dado inserido em uma página para que o navegador a renderize. [índice]

d

Tags fundamentais

As tags fundamentais de um documento html são as seguintes: HTML, Head e Body. Html é a tag principal, que caracteriza o documento e onde todas as outras tags se alojam. A tag Head por sua vez é uma espécie de meta informação, dentro dela se coloca dados que não serão exibidos ao usuário, mas ao navegador. Por fim, a tag Body que marca a parte que é apresentada ao usuário, os textos, imagens e qualquer outra coisa que pode ser percebido estão. [índice]

Tags de Texto

Entre as tags de texto estão a tag p: esta representa a tag de paragrafo. Também existe a familia das tags h, que vão de h1 à h6. [índice]

Links

Para criar conexões, ancoras entre as páginas ou mesmo entre seções das mesmas pdemos usar a tag a, que define os links. [índice]
